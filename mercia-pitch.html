Silverwings started as a benefits claims service. That service is now cashflow-positive, approaching £100K/month, and growing. But the real value isn't the claims — it's the technology stack we built to automate them.

Every tool we built to run Silverwings faster — the CRM, the code scanner, the voice AI agents, the compliance engine — turns out to be a standalone SaaS product that other businesses need.

The thesis: Silverwings Benefits generates cashflow. The Venture Foundry spins out SaaS products from the technology stack. Each spin-out has its own market, its own revenue, and its own exit potential. The £750K accelerates the spin-outs from "built" to "selling."

What Chris Saw in the Meeting

Three things lit up in the meeting — all three are products we've already built:

1. Thuban — AI Code Verification

Chris mentioned that Mercia literally paid for code due diligence yesterday on a portfolio investment. Thuban does exactly that — automated, instant, at a fraction of the cost.

  • Already launched. Live on npm. Stripe payments wired. Website live at thuban.dev.
  • Pricing: Free / $9 Pro / $49 Team / $299 Enterprise + $299-$999 one-off investor reports
  • 10 languages, 69 security rules, AI hallucination detection — the only scanner purpose-built for AI-generated code
  • Direct relevance to Mercia: Every investment they make requires code due diligence. Thuban provides it in minutes, not weeks.

Status: LAUNCHED — needs marketing push, not more development.

2. AI Sales Agents — The SDR Killer

Mercia's portfolio companies burn through SDRs in an average of 14 months. Our voice AI agents qualify leads at machine speed, 24/7, with zero churn.

  • Already built. Triage agent running in production for Silverwings. LiveKit + OpenAI Realtime + Twilio integration.
  • Speed-to-lead: Responds in under 2 seconds. Qualifies in under 3 minutes. Hands off warm leads to humans.
  • Cost: ~£0.30 per qualified conversation vs ~£25-50 per SDR conversation
  • Direct relevance to Mercia: Deployable to any portfolio company with an inbound sales funnel. Immediate portfolio value creation.

Status: PRODUCTION-READY — running internally, needs productisation for external sale.

3. Navisynth — Voice Agent Compiler

The platform that builds the voice agents. Feed it a company's real call recordings, and it compiles a voice agent trained on their actual data, tone, and processes.

  • Full SaaS architecture: FastAPI backend, Next.js dashboard, microservices, Terraform infrastructure, CI/CD
  • The meta-product: Instead of selling one agent at a time, sell the platform that creates agents
  • Market: Every contact centre, every sales team, every support desk

Status: BUILDING — 2-3 weeks from live testing with real data.
